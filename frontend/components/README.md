# components/

Reusable UI components for the ChainEquity dashboard.

## Structure

- `ui/` - Base UI components (buttons, cards, tables, forms)
- `wallet/` - Wallet connection and management components
- `token/` - Token-specific components (minting, transfers)
- `captable/` - Cap-table viewer and export components
- `corporate/` - Corporate actions components (splits, symbol changes)
- `layout/` - Layout components (header, sidebar, footer)

## Usage

```typescript
import { Button } from '@/components/ui/button';
import { WalletConnect } from '@/components/wallet/WalletConnect';
```
